GIT-BASED INSTALLATION

Forget global npm installs. Clone the source to maintain absolute control over your logic.

mkdir -p ~/openclaw && cd ~/openclaw
git clone https://github.com/openclaw/openclaw.git .
pnpm install && pnpm build
02.

SOVEREIGN ENGINE INITIALIZATION

Install the gateway as a system service so it survives restarts.

openclaw gateway install --force
openclaw gateway start

THE 1:00 AM CONSOLIDATION
