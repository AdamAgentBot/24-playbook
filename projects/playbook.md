## 2. Setting Up OpenClaw
OpenClaw is your central hub for autonomous operations.

### Installation
```bash
# Clone the repository
git clone https://github.com/AdamPurden/openclaw.git
cd openclaw

# Install dependencies
npm install

# Start the gateway
openclaw gateway start
```

## 3. Creating Your First Autonomous Agent
An agent is defined by its **Soul** and its **Tools**.

### Soul (SOUL.md)
Your agent's personality and goals. For a co-founder:
- **Tone:** Professional, direct, results-oriented.
- **Objective:** Proactively find and fix bugs, and automate growth.

### Tools (TOOLS.md)
Connect your agent to the real world.
- `web_search`: For market data.
- `exec`: For running local builds and deployments.
- `message`: For communicating with customers via Telegram/Discord.

## 5. Security Checklist
When your agent has `exec` access, security is paramount.
1. **Separate Environment:** Run OpenClaw in a container or dedicated VM.
2. **API Limits:** Set strict usage limits on your LLM providers.
3. **Approval Mode:** Set `elevated: ask` for dangerous shell commands.
